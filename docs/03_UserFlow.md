# BookDNA AI — User Flow

## 1. Primary User Journey

```text
Landing Page
   ↓
Sign Up / Login
   ↓
Reader DNA Onboarding Quiz
   ↓
Choose Theme Preference
   ↓
Personal Dashboard
   ↓
Personalized Recommendations
   ↓
Book Details
   ↓
Add to Wishlist / Currently Reading / Completed
   ↓
Rate Book and Give Feedback
   ↓
Reader DNA Updates
   ↓
New, Improved Recommendations
```

## 2. Landing Page Flow

```text
Landing Page
   ↓
Explore BookDNA AI Features
   ↓
Click “Discover My Reader DNA”
   ↓
Sign Up
```

The landing page should introduce the platform, explain Reader DNA, show personalized recommendation examples, and encourage visitors to create an account.

## 3. Onboarding Flow

```text
Create Account
   ↓
Choose Favorite Genres
   ↓
Add Favorite Books
   ↓
Select Preferred Reading Moods
   ↓
Choose Reading Preferences
   ↓
Generate Reader DNA
   ↓
Choose Theme
   ↓
Open Dashboard
```

### Onboarding Inputs

* Favorite genres
* Favorite books
* Favorite authors
* Preferred moods
* Preferred pacing: slow, medium, or fast
* Preferred endings: happy, sad, open, or plot twist
* Interest in romance, mystery, fantasy, horror, science fiction, and other genres

## 4. Recommendation Flow

```text
Dashboard
   ↓
Recommended For You
   ↓
Open Book Details
   ↓
View “Why This Was Recommended”
   ↓
Add to Wishlist / Currently Reading
   ↓
Recommendation Feedback Stored
```

### Recommendation Feedback Signals

* Book opened
* Book saved
* Book added to wishlist
* Book marked as currently reading
* Book marked as completed
* User rating
* Recommendation ignored

These signals will later help improve personalization.
