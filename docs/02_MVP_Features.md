# BookDNA AI — MVP Features

## Goal of the MVP

The first version of BookDNA AI will focus on helping users discover books through personalized recommendations while proving the core AI, product, and user-experience concepts.

The MVP will avoid building every planned feature at once. Advanced social, gamification, and scanner features will be added later.

## Must-Have Features

### 1. User Authentication

* Sign up
* Login
* Logout
* Basic user profile

### 2. Reader Onboarding

* Select favorite genres
* Add favorite books
* Select preferred moods and reading styles
* Generate an initial Reader DNA profile

### 3. Book Discovery

* Search books by title, author, or genre
* View book cover, author, description, genres, rating, and tags
* Browse trending and popular books

### 4. Personalized Recommendations

* Recommend books based on favorite genres, books, and reading preferences
* Show similar books
* Display a recommendation score
* Show a “Why this was recommended” explanation

### 5. Personal Library

* Add books to Wishlist
* Mark books as Currently Reading
* Mark books as Completed
* Give a rating after completing a book

### 6. Reader DNA

* Display genre and preference percentages
* Show a primary reading personality
* Update the profile after user interactions and ratings

### 7. Theme Preferences

* Light mode
* Dark mode
* One mood-based theme, such as Cozy Reader or Midnight Library

### 8. Basic Gamification

* Reading streak
* Books completed count
* One or two starter achievements

### 9. Basic Analytics

* User dashboard with books read, wishlist count, streak, and favorite genres
* Admin view for popular genres, searches, and books

## Out of Scope for MVP

These features are planned for later versions:

* AI chat companion
* OCR/ISBN book scanner
* Social following and reader communities
* Reader room
* Ambient music
* Seasonal events
* Full mascot evolution system
* Advanced collaborative filtering
* Mobile application
* Publisher analytics

## MVP Success Criteria

The MVP is successful if a user can:

1. Create an account.
2. Complete onboarding.
3. Receive personalized book recommendations.
4. Understand why a book was recommended.
5. Save and track books in a personal library.
6. View their Reader DNA and reading progress.
7. Use the application comfortably on desktop and mobile.
